# Chatbot Webhook API

## 🚀 Overview
> A simple Spring Boot webhook API demonstrating clean backend architecture. It exposes a REST API that receives a message and returns a predefined response.

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Maven

## 📡 API Endpoint
**POST** `/webhook`

### Request
{
  "message": "hi"
}

### Response
{
  "reply": "Hello"
}

## 💡 Features
- REST API with JSON request/response  
- Clean architecture (Controller + Service)  
- Basic chatbot logic (hi, hello, bye)  
- Logging implemented
